# Brochures / Fichas técnicas

Los botones "Descargar ficha técnica" de cada producto apuntan a esta carpeta.
Para activarlos, subir los PDF oficiales con estos nombres exactos:

| Producto        | Archivo esperado            |
|-----------------|-----------------------------|
| Tega Crane      | `tega-crane.pdf`            |
| Power Lift      | `power-lift.pdf`            |
| Ultra Lift      | `ultra-lift.pdf`            |
| Anti Explosivo  | `anti-explosivo.pdf`        |
| Tega Chain      | `tega-chain.pdf`            |
| Tega Kit        | `tega-kit.pdf`              |
| Radiolift       | `radiolift.pdf`             |
| MoviCar         | `movicar.pdf`               |
| Tega Service    | `tega-service.pdf`          |

Mientras no exista el PDF, el botón queda visible pero el enlace dará 404.
Si se prefiere ocultar el botón hasta tener el PDF, quitar la propiedad
`brochure` del producto correspondiente en el objeto `PROD` dentro de `index.html`.
